# Frontend Parfum

Frontend ini disetel untuk local development saja.

## Menjalankan

Jalankan dari folder `frontend-parfum`:

```bash
npm run dev
```

Frontend akan memakai backend lokal:

```text
http://127.0.0.1:8000/api
```

Pastikan backend berjalan dengan:

```bash
php artisan serve
```

Tidak ada workflow build/preview/deploy di project ini.
